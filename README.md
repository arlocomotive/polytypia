# LuaCATS type definitions for Polytoria 1.0

Notice
------
This is not fully complete and was pretty sloppily put together. Some parts still need a lot of work. This is just "good enough" for scripting a game.

###### This is not fully "required", but this project also aims to document literally EVERY part of Polytoria 1.0, even down to its obscure and hidden classes.

How to use
----------
Visual Studio Code (and [this Lua extension](https://open-vsx.org/extension/sumneko/lua)) were intended to be used, but any code editor with a Lua language server that supports LuaCATS should work just fine. For VSCode, you should just be able to open a script file while navigated to Polytypia and it'll give you type definitions.
